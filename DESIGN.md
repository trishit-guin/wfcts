# Design System Theme

## Overview
This document outlines the core visual theme for our design system, reflecting our brand's aesthetic and functional requirements.

## Color Palette

### Color Mode
The theme is set to `light` mode.

### Primary Colors
- **Primary Color:** `#1E3A8A` - A deep blue, used for primary actions, branding, and key interactive elements.
- **Secondary Color:** `#0D9488` - A vibrant teal, complementing the primary color and used for secondary actions or accents.
- **Tertiary Color:** (Not explicitly defined in current theme, consider adding if needed for additional accents.)
- **Neutral Color:** `#F3F4F6` - A very light grey, serving as a base for backgrounds and non-chromatic UI elements.

## Typography

### Headline Font
`Manrope` - Chosen for its modern and geometric feel, suitable for headlines and prominent text.

### Body Font
`Inter` - A highly readable and versatile sans-serif font, used for general body text to ensure clarity across all content.

### Label Font
`Inter` - Consistent with the body font for labels, ensuring readability and visual harmony in forms and interactive elements.

## Shape and Spacing

### Roundedness
`2` - Elements feature a moderate level of roundedness, providing a balanced and friendly aesthetic without being overly soft or sharp.

### Spacing
`2` - The design employs normal spacing, offering a comfortable visual balance between elements and ensuring good readability and usability.
